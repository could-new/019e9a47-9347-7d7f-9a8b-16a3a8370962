# بوابة عشتار الأمنية

تطبيق احترافي لنظام الحضور والانصراف المخصص لشركة بوابة عشتار الأمنية. يعمل التطبيق على منصات متعددة ويوفر واجهة مستخدم عربية بالكامل لدعم إدارة الموظفين، المشرفين، وعمليات الحضور والانصراف باستخدام تقنية مسح الـ QR Code.

## المميزات
- **تسجيل الدخول**: نظام تسجيل دخول آمن للمدراء والمشرفين والموظفين.
- **مسح QR Code**: يقوم المشرف أو المدير بمسح رمز الـ QR الخاص بالموظف لتسجيل حضوره أو انصرافه.
- **إدارة الموظفين والمشرفين**: لوحة تحكم متكاملة للمدير لإدارة المستخدمين وصلاحياتهم.
- **واجهة عربية وتصميم عصري**: تصميم احترافي يدعم الوضع الليلي (Dark Mode).

## التقنيات المستخدمة
- **Flutter**: لبناء واجهات التطبيق وتوفير الدعم لجميع المنصات.
- **Supabase**: لخدمات قواعد البيانات والمصادقة وإدارة السجلات (حضور وانصراف، مستخدمين، رموز QR).

## طريقة التشغيل
1. تأكد من تثبيت Flutter SDK.
2. قم بتشغيل `flutter pub get` لتنزيل الاعتماديات.
3. قم بتشغيل التطبيق باستخدام `flutter run`.
